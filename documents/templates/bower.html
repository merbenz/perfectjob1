Bower can manage components that contain HTML, CSS, JavaScript, fonts or even image files. Bower doesn’t concatenate or minify code or do anything else - it just installs the right versions of the packages you need and their dependencies.


Installing Bower

To install gulp first you must have NodeJs installed, NodeJS will have npm.

And then install bower globally by running the following command npm install -g bower


Installing Package

Install packages with bower install. Bower installs packages to bower_components/.

bower install PACKAGE_NAME

A package can be a GitHub shorthand, a Git endpoint, a URL, and more. Read more about bower install.

Examples

# installs the project dependencies listed in bower.json
bower install

# registered package
bower install jquery

# GitHub shorthand
bower install desandro/masonry

# Git endpoint
bower install git://github.com/user/package.git

# URL
bower install http://example.com/script.js

Save Packages

Create a bower.json file for your package with bower init.

Then save new dependencies to your bower.json with;

bower install PACKAGE_NAME --save
