In one word: automation. The less work you have to do when performing repetitive tasks like minification, compilation, unit testing, linting, etc, the easier your job becomes. After you've configured it through a Gruntfile. AdminBSB - Sensitive supports Grunt to make it even easier for developers, this allows us to implement a powerful command line interface for pages.


Installing Grunt

To install gulp first you must have NodeJs installed, NodeJS will have npm.

And then please follow these instrucations:

  1. Install grunt globally by running the following command npm install -g grunt-cli
  2. In you downloaded package, navigate to grunt/ directory and copy both package.json , Gruntfile.js to your project root.
  3. Navigate to the root directory of your project, then run npm install.

Commands

grunt build

This will automatically compile & minify your assets

grunt watch

This will automaically compile the pages Sass files on save

grunt sass

This will compile the sass files


Troubleshooting

Should you encounter problems with installing dependencies or running Grunt commands, first delete the /node_modules/ directory generated by npm. Then, rerun npm install.
